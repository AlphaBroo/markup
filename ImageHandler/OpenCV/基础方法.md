# 基础方法

## 安装

```shell
pip install python-opencv
```

## 使用

### 读取创建

```python
import cv2 as cv

src = cv.imread("D:/vcprojects/images/demo.png"  # 读取图像
cv.namedWindow("input image", cv.WINDOW_AUTOSIZE)  # 创建一个重新命名的窗口
cv.imshow("input image", src)  # 展示图像
cv.waitKey(0)  # 延迟等待
cv.destroyAllWindows()
```
