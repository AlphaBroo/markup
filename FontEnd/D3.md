# D3

D3(Data Drivern Documents)即数据驱动文档，是JS图形可视化工具，可以实现更加灵活且富有变化的图形，D3使用的DOM元素大多属于svg元素，如line,rect,circle等，可以放大而不失真。

核心思想

```
数据驱动文档，通过数据来控制DOM元素的内容和外观
```

使用流程

```
为DOM元素绑定数据
利用数据确定DOM元素内容和外观等属性
当数据发生变化时，相应地更新DOM元素
```
